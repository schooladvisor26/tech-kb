VS Code
CTRL+SHIFT+P (one shortcut to open "everything")
Try searching view toggle.

Line Actions
Ctrl+C copies an entire line when no text is selected.
Alt+UpArrow and Alt+DownArrow moves lines
Ctrl+Shift+K deletes lines
Ctrl+/ to comment
Rename Factoring
F2 or Right-click rename symbol (Work as opposed to Book in list)
Formatting
Ctrl+Shift+I to format conten for the entire document
Ctrl+K Ctrl+F for hte selection
Tip: Additional options are available in the extensions gallery or can be setup via settings.
Try editor.FormatOnSave.
Code Folding
It can be helpful to "collapse" sections of code in large files to "increase readability.
Ctrl+Shift+[ to fold and Ctrl+Shift+] to unfold sections of code.
Ctrl+K Ctrl+0 or Ctrl+K Ctrl+J to unfold all.
Tip: Folding is based on indentation so it applies to all languages.
That means you can just indent your code and fold "certain number of levels" with the shortcuts Ctrl+K Ctrl+1 through Ctrl+K Ctrl+5.

Instructions for Data backups using Backup Tool (SOP)
1. Insert USB flash drive
2. Launch backup tool (not Timeshift)
3. Click on "Back Up Now..." under Personal data
4. Choose where you want the backup file (choose the USB Flash drive)
5. Exclude any directories you need to exclude like maybe VirtualBox VMs.
6. On the next screen, include any directories normally hidden that you need to include (I never incluude any).
7. Click OK or Backup or whatever it says.
8. I've yet to Restore any personal data (backups) which is an option on the first screen.

Installing flatpaks in Ubuntu MATE 22.04
You may not see them in the Boutique.
Visit flathub.
Download the file of your choice.
There may not be a way to just install the file like you can with Gdebi.
Open the folder where the file is in in terminal.
Run the following command:
 flatpak install packagename
Replace packagename with the name of the file.
